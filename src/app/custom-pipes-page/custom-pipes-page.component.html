custom pipes can be generated by using Angular cli command 'ng generate pipe pipe_name'.
It contains '@pipe' decorator where name of pipe is mentioned and we should use that pipe name where ever we need to use the pipe.
That pipe should be imported to angular module so that it can be used cross the project templates.



Let us define a custom pipe which identifies the given number is a signed decimal number or not:

The below input field will accepts user input and the pipe will identifies that given number is accepted or not.
Accepted number is signed or unsigned decimal or integer numbers it wont accept any alpha characters or special symbols except '+' or '-' that too at the beginning of the input.
If user enters any unaccepted chars below the field it will be shown as 'Not a signed decimal number' highlighted in red color.
If user removes the unaccepted character again pipe will display the valid input given by the user.
